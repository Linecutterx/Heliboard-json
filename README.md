https://github.com/HeliBorg/HeliBoard is an android keyboard that can be configured using json files to change the key layout & the popup keys related to them.

https://sangaline.com/post/finding-an-optimal-keyboard-layout-for-swype/ produced a keyboard layout that optimises Swype accuracy (in 2015) & I was curious about how that would look/work in real life.

I have an idea that I'd like to reproduce the original (& sadly deceased) Swype keyboard - which I have too much muscle memory developed to discard.

These are the results...

Import them as described here: https://github.com/HeliBorg/HeliBoard/wiki/2.-Layouts

& maybe edit/make your own here: https://roccobot.github.io/HeliBoard-RLM/
